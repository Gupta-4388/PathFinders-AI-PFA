# **App Name**: PathFinder AI

## Core Features:

- Resume Analyzer: Upload, parse, and analyze resumes to identify skills, projects, and mistakes, providing AI-driven improvement insights and skill summaries. Highlights missing industry-standard skills and recommends learning paths.
- AI Mentor: Provide personalized career guidance, mentorship suggestions, skill growth roadmaps, and job market insights using an AI-powered chatbot. Recommends relevant learning resources such as YouTube videos, course links, and websites. Prioritizes free certifications and includes a progress tracker for skills and certifications. Display specialized mentors based on chosen career paths, showing certifications and offering connection options via LinkedIn and Naukri. Focuses on free certifications first. Supports multiple languages with dynamic translation.
- Career Dashboard: Recommend roles based on resume insights and user preferences. The dashboard will also visualize skill gaps, growth paths, salary ranges, and a learning timeline, with shortcuts for applying to jobs, starting courses, or finding mentors.
- Job Market Trends: Visualize current job trends from both India and outside India using dynamic charts and graphs, highlighting in-demand skills, salary benchmarks, and hiring patterns. Includes integration with live or placeholder APIs.
- Mock Interview Simulator: Simulate domain-specific interviews with AI-generated questions, voice and camera support, and real-time analysis of tone, clarity, and content. Provides an improvement tool tips and a final score evaluating relevance, correctness, fluency and confidence.

## Style Guidelines:

- Primary color: Dark Grey (#212529) provides a professional and modern look.
- Background color: Very Dark Grey (#121212) offers high contrast and reduces eye strain.
- Accent color: Vibrant Green (#198754) for CTAs and key interactive elements. The logo will also use this green.
- Font: 'Inter', a grotesque-style sans-serif, provides a modern, machined, objective, neutral look suitable for both headlines and body text.
- Use modern line icons throughout the dashboard to maintain a clean and intuitive user interface. Use the logo provided.
- Employ a responsive design to ensure the application works seamlessly on desktop and mobile devices. Utilize a dashboard UI for easy navigation and information presentation.
- Incorporate smooth transitions, hover effects, and minimal animations to enhance user experience without overwhelming the interface.
